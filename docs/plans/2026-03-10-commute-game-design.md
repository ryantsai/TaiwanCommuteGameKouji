# 機車通勤遊戲 設計文件

**日期：** 2026-03-10
**平台：** 單一 HTML 檔案（Canvas 2D API，原生 JS）
**美術風格：** 像素風（16×16px 格）
**視角：** 俯視 2D 地圖，垂直捲動

---

## 遊戲概念

玩家騎機車從家出發通勤到公司，模擬台灣一週（5天）的通勤生活。每天地圖略有不同，會遇到各種障礙與隨機事件。週五結局依總分決定。

---

## 遊戲流程

```
標題畫面 → 每日簡報（天氣/事件預告）→ 通勤關卡 → 結算畫面 → 下一天
共 5 天（週一～週五），第 5 天有多結局
```

### 每日關卡
- 地圖由上往下捲動，玩家機車固定在畫面下 1/3
- 頂部顯示「通勤進度條」
- 生命值歸零視為該天遲到（扣分懲罰，不 Game Over）

---

## 計分系統

| 事件 | 分數 |
|------|------|
| 平安抵達 | +100 |
| 時間獎勵（越快越多） | +0~50 |
| 被撞 | -20 |
| 撞上障礙物 | -10 |
| 遲到（生命值歸零） | -30 |

生命值：每天 3 條命，撞擊扣 1 命。

---

## 障礙物系統

### 靜態障礙
| 障礙 | 效果 |
|------|------|
| 違規停車（紅色車） | 佔用車道，需繞行 |
| 坑洞 | 減速 + 閃避不及扣血 |
| 路邊攤 | 縮窄道路 |
| 施工區域 | 大範圍封路，強制換道 |

### 動態障礙
| 障礙 | 行為 |
|------|------|
| 移動車輛 | 同向慢速行駛，需超車 |
| 突然開車門 | 停放的車突然開門 |
| 衝出來的機車 | 從巷口高速衝出 |
| 行人過馬路 | 橫向穿越 |

---

## 天氣系統

每天開始時隨機決定天氣：
- ☀️ **晴天**：正常操控
- 🌧️ **雨天**：滑動慣性增加，煞車距離變長
- 🌫️ **起霧**：視野縮短，障礙出現更突然

---

## 隨機事件

通勤途中隨機觸發：
- 🚨 **警察開罰單**：強制停車 3 秒
- 🚑 **救護車通過**：所有車輛靠邊停
- 📱 **導航沒訊號**：小地圖消失 5 秒

---

## 操控

| 按鍵 | 動作 |
|------|------|
| ← → 方向鍵 | 左右移動換道 |
| ↑ 方向鍵 | 加速 |
| ↓ 方向鍵 | 煞車/減速 |
| Space | 按喇叭（讓行人/車輛短暫停頓）|

---

## 技術規格

- **畫面尺寸**：480×640px（直式）
- **像素格**：16×16px
- **渲染**：Canvas 2D API
- **地圖捲動**：玩家固定，背景向下捲動
- **碰撞偵測**：AABB（軸對齊包圍盒）
- **UI**：頂部顯示天數、生命值、分數、時間

### 像素美術
- 玩家機車：藍色 16×24px
- 一般車輛：紅/黃/白色方塊車
- 障礙物用顏色區分
- 道路：深灰，車道白虛線，人行道淺灰
